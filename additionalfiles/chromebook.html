Chromebook is a product of google which tries to provide a fully functioning machine with low hardware requirements. This amazing thing happens due to the light weight Chrome Os. It has all features of a android tablet but comes with extra asset of desktop capabilities,thus becomes an effivient coding machine.

For Coding:-

Code Editors:

Sublime Text
Vim(best as it works on terminal also saves space )

App Making:

Android Studio(comes Inbuilt)

IDEs:

NetBeans
Pydroid3


Terminal is provided in the form of Linux Beta. Allows us to install Linux distros thus giving the freedom to use linux.







For Video Editing:

Kinemaster

Lightworks(works on linux distro).

For Office:

WPS Office

Google Office

Libre Office(Linux Distro).




Chromebook is an efficient coding machine but the addition of Linux gives it that final touch of a budget machine capable of being your primary computer
